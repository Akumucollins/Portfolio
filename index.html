AkanName

Akan-Name Generator is a web application that takes a user's birthday and calculates the day of the week they were born and then depending on their gender outputs their Akan Name.

QuotesApp

Quotes App is web application that allows user's to add their own quotes by inputing their username, quote category, their favorite quote, authors' name and date.The web application also allows the users to like , dislike and delete the quote

GitUserApp

GitUser App is an angular website that allows users to enter a GitHub username into a search input, submit it, and see names, nicknames, locations, email account, followers, people they are following, biography and descriptions of that person's public repositories.

NewsHeadlines

News Headlines is a web application that displays a list of all various news sources and users can view news articles and their categories. This website makes use of News API to retrieve top news articles from different news sources with relative categories.

BlogsWebsite

Blog Website is a django web application that allows users to can create and share your opinions and other users can read and comment on them. Blogs also has random quotes that inspire the users.

PhotoGallery

Photo Gallery is a django web application for a personal gallery that allows an admin to display thier photos for users to see and be able to share by coping the image link.

Neighbourhood

Neighbourhood is a web application where a user must signup first, be able to join a hood owned by the hood admin, and once you join a hood, one can see businesses and posts in only that hood they belong to.

CaloriesTracker

Calorie Counter is a web application that allows a user the capability of calculating the number of calories he/she should be consuming based on their height and weight.The user can also keep track of the number of calories that they have burned in a day based on the amount of exercise.
